-------НАЧАЛО РАБОТЫ--------
cd 'ссылка' - переход в папку
git clone ссылка - подключение к репрезиторию
git status - проверка файлов в папке

-------РАЗМЕЩЕНИЕ ФАЙЛОВ-------
git add . - отслеживать файлы
git commit -m "комментарий" - отправить на локальный сервер
git push - отправить на GitHub

--------ВЕТКИ-----------
git branch - все ветки
git branch ch1 - создать ветку ch1
git checkout ch1 - перейти на ветку ch1

git push --set-upstream origin ch1 - флаг для первой отправки с новосозданной ветки ch1

------СЛИЯНИЕ ВЕТОК------
git merge ch1 (если в main) - слияние веток

-----КОНФЛИКТЫ------
при возникновении конфликта - заменить 
файлы с конфликтным содержимым, на 
содержимое, которое вам нужно по итогу
